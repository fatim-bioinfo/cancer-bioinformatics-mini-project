# Mini Cancer Bioinformatics Project

This project analyzes Gene_A expression, patient survival status,
and drug response using Python.

## Methods
- Group comparison (Alive vs Dead)
- Boxplot visualization
- Scatter plot analysis
- Correlation calculation

## Results
Gene_A expression was higher in the Dead group.
A negative correlation was observed between Gene_A and Drug_Response.

## Tools Used
- Python
- pandas
- matplotlib
